# Supported PDDL Features

Tyr supports the following Planning Domain Definition Language (PDDL) features:

  - `:action-costs`
  - `:conditional-effects`
  - `:derived-predicates`
  - `:disjunctive-preconditions`
  - `:equality`
  - `:existential-preconditions`
  - `:negative-preconditions`
  - `:numeric-fluents`
  - `:strips`
  - `:typing`
  - `:universal-preconditions`
